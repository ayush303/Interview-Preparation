# Interview Preparation

A structured collection of problems and concepts for technical interview preparation, covering **Low-Level Design (LLD)**, **High-Level Design (HLD)**, and **Data Structures & Algorithms (DSA)**.

---

## Contents

### Low-Level Design (LLD)
Object-oriented design problems focused on class design, design patterns, and system modeling.

- Class diagrams and entity relationships
- SOLID principles in practice
- Common design patterns (Factory, Strategy, Observer, etc.)
- Real-world system designs (Parking Lot, Library Management, Ride Sharing, etc.)

### High-Level Design (HLD)
System design concepts for large-scale distributed systems.

- Scalability, availability, and consistency trade-offs
- Databases: SQL vs NoSQL, sharding, replication
- Caching, load balancing, message queues
- Common system designs (URL Shortener, News Feed, Rate Limiter, etc.)

### Data Structures & Algorithms (DSA)
Topic-wise problem sets with solutions.

| Topic | Description |
|-------|-------------|
| Arrays & Strings | Sliding window, two pointers, prefix sums |
| Linked Lists | Reversal, cycle detection, merging |
| Trees & Graphs | BFS, DFS, shortest path, topological sort |
| Dynamic Programming | Memoization, tabulation, classic patterns |
| Heaps & Priority Queues | Top-K problems, scheduling |
| Backtracking | Permutations, combinations, constraint problems |
| Greedy | Interval scheduling, optimization |
| Bit Manipulation | Tricks and common patterns |
